Project Description

The project aimed to develop a web app featuring an integrated live chat system, including an AI chatbot that can be embedded into any website. The AI chatbot is designed to handle initial interactions with users and includes a live agent handoff feature, allowing users to be connected to a human agent when needed. Once an agent joins the chat, the AI chatbot will cease interaction, and an agent dashboard will become available for the agent to manage the conversation. If the agent exits the chat, users will be informed, and the AI chatbot will resume interaction to continue assisting the user.

The Outcome

The outcome of the project was successful, as I was able to implement the AI chatbot and live chat system effectively. However, due to limitations with Google's API, the feature currently supports only Android devices. Unfortunately, this means that the chatbot is not yet functional for iOS or web platforms. I have contacted Google, and they informed me that they are working on updates to the API to support iOS and web. We are now awaiting their finalization of these updates to enable full cross-platform functionality.

Project Description

Surge is a mobile application designed to streamline the process of opening a new bank account without requiring a visit to a physical bank. To create an account, users need a valid Ghana Card and must provide a selfie along with other essential information. This data is sent to the National Identification Authority for verification. The verification process typically takes no more than two minutes, after which the account is created and pending final approval from the bank. Once the bank has approved your details, you can use Surge to send and receive money directly through the app.

The Outcome

So far, the registration and account verification features are fully operational. However, the transactions component is still under development, as it requires additional security measures and protocols. This functionality will be included in version 2 of the app. Version 1 is complete and includes features such as balance viewing, with the transaction capabilities being addressed in the upcoming update.

Project Description

The project involved the development of a comprehensive web and mobile application designed to track and monitor pet health. The app integrates with a specialized collar device to provide real-time data on various health metrics, including heart rate and activity levels. Users can access detailed health insights, receive timely notifications, and benefit from personalized health tips directly through the app. It also features robust pet profile management, allowing owners to maintain detailed records of their pets' health and well-being.

Additionally, the app includes an interactive map of nearby veterinary clinics, enabling users to quickly locate and navigate to the nearest vet in case of an emergency. This feature ensures that pet owners can promptly address their pets' medical needs, providing peace of mind and enhancing the overall pet care experience. The combination of real-time health tracking, educational resources, and convenient access to veterinary services offers a holistic approach to managing and improving pet health.

The Outcome

The project is largely complete, with all core functionalities built and operational. The web and mobile apps are fully functional, providing users with real-time health insights, notifications, and veterinary location mapping. Currently, I am focusing on integrating the pet collar with Firebase for data transmission. While I have successfully transmitted data from the collar using Python based on health metrics readings, there are still some configuration settings that need to be finalized. The collar will send data tied to each pet's unique serial number, which users will need to input to link their pets to their profiles. This final step is crucial for ensuring accurate and personalized health tracking.
